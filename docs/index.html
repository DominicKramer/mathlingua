MathLingua

The MathLingua language is used to encode mathematical definitions, theorems, axioms, and conjectures.

The goal of the MathLingua system is to have a central location of mathematical results that can be searched by, contributed to, and built on by anyone anywhere.

Filter by keywords to find what is already encoded in MathLingua. For example, filter using the keyword "group".

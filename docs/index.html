MathLingua Playground

The editor below can be used to write MathLingua encoded mathematics and try the MathLingua language.

The example below shows some of the power of MathLingua. First, what it means for an operator to be commutative is defined. Next, an Abelian group is defined, and a theorem is described that states that all cyclic groups are Abelian.

When the Expand button is pressed, all of the definitions and results will be expanded with all definitions inlined. This demonstrates how definitions and results interact with each other and how MathLingua can be used to analyze mathematical results.

In particular, the definition of a commutative operator will be inlined in the definition of an Abelian group, and the definition of an Abelian group will be inlined in the theorem. It is fine that a cycle group is not defined, but, because of that, it is not expanded.

[\commutative:on{A}] Defines: * assuming: 'A is \set' means: . for: a, b where: 'a, b \in A' then: 'a * b = b * a' [\abelian.group] Defines: G := (X, *, e) means: . 'G is \group' . '* is \commutative:on{X}' Result: . for: G where: 'G is \cyclic.group' then: 'G is \abelian.group'

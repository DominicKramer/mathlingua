MathLingua Playground

The editor below can be used to write MathLingua encoded mathematics and try the MathLingua language.

The examples below shows some of the features of MathLingua. First, it is informally defined (as denoted by the use of double quotes) what it means by x \in X. Next, there is a definition of a continuous function. Last, there is a result that states that a differentiable function on a set is continuous on that set.

When the Expand button is pressed, all of the definitions and results will be expanded with all definitions inlined. This demonstrates how definitions and results interact with each other and how MathLingua can be used to analyze mathematical results.

[a \in A] Represents: assuming: 'A is \set' that: "$%a$ is an element of $%A$." [\continuous.function:on{X}] Defines: f assuming: 'X is \set' means: . 'f is \function:on{X}' . for: x where: 'x \in X' then: . '\limit[t]_{x}{f(t)} = f(x)' Result: . for: g, Y where: . 'Y is \set' . 'g is \function:on{Y}' then: . if: 'g is \differentiable \function:on{Y}' then: 'g is \continuous \function:on{Y}'

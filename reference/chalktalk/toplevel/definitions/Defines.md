# Definition

A `Defines:` form is used to define a mathematical object, such as a set, function, or group, etc.

```yaml
[<id>]
Defines: <target>
when?: <is-statement>+
means?: <clause|is-statement>+
evaluated?: <evaluated-clause>+
using?: <using-clause>+
written: <text>
Metadata?: <metadata-form>
```

# Expression

An expression is either an `is` statement, an `equals` statement, a `colon-equals` statement, a `double-colon-equals` statement or a sequence of terms and infix operators.

## Example

```latex
x is \set
f(x) := x
f(g(x?, y?)) ::= x + y
x + y = 0
\sin(x)^2 + \cos(x)^2 = 1
\integral[x]_{0}^1{f(x)}
```
